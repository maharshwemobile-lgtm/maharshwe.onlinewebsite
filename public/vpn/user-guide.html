User Guide

Use this guide when installing MaharShwe VPN, adding keys, scanning QR codes, and choosing free or paid servers.

1. Install

Download the APK from maharshwe.online/vpn, install it, and open MaharShwe VPN.

2. Allow permissions

Allow VPN permission when Android asks. Allow camera permission only if you want to scan a QR key. Allow notification permission to receive update messages.

3. Add an access key

Tap the plus button, paste your Outline access key, or choose Scan Key and scan the QR code.

4. Connect

Choose a server and tap Connect. If the Android VPN confirmation appears, approve it.

5. Free server notice

Free servers are shared. They may be slower or unstable. The app may show ads and reminder messages while using free servers.

6. Paid/private key

For a more stable connection, buy a private VPN key from support. Paid/private keys do not show the free server warning.
